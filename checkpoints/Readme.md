# Model Checkpoints

Download pre-trained weights and place them in this directory.

## Required Files

| File | Model | Size | Description |
|------|-------|------|-------------|
| `zoomnext_b4.pth` | ZoomNeXt EfficientNet-B4 | ~80MB | Camouflaged object detection |
| `swin_v2_small_best.pth` | Swin V2-Small | ~200MB | Cam/NoCam classification |

## ZoomNeXt Weights

Download from: [ZoomNeXt GitHub](https://github.com/lartpang/ZoomNeXt)

Select the EfficientNet-B4 backbone variant.

## Swin V2-Small Weights

Trained on our custom dataset. Available upon request or train yourself using the provided notebook.
